# 🧩 coding-challenges

Repositório de resoluções de desafios práticos de programação feitos por **Mikael Kobama** como parte da jornada de evolução técnica contínua.

> Para desafios mais complexos e aprofundados, veja o repositório [edge-cases](https://github.com/MikaelKobama/edge-cases).

---

## 🎯 Objetivo

Registrar evolução consistente através da prática diária em diferentes plataformas, linguagens e áreas do desenvolvimento.
